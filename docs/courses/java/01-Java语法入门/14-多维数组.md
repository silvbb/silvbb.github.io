# 多维数组

## 前言

**C：** 上一篇，我们学习了数组的概念，并掌握了它的定义及基本使用方法。

下面，我们再来看一个案例需求。

**案例需求：已知有3个班级各5名学员，请计算各个班级的总成绩。** 

根据需求及经验来分析，如果我们只是要计算单个班级的学生总成绩，那么定义一个普通数组，来存储这些成绩信息，再去计算即可。

```java
// 存储一个班成绩
double[] scores = new double[5];
// ...
```
